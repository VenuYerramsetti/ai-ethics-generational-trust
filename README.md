# AI Ethics: Generational Trust in Synthetic Media

## Overview
This project explores how different generations perceive AI-generated content and the ethical risks posed by synthetic media such as deepfakes, AI-generated voices, and text.

## Motivation
"My parents could not distinguish between real and AI-generated content — this raised critical concerns about trust, misinformation, and digital safety."

## Key Contributions
- Research paper on generational vulnerability to AI content
- Real-world examples of synthetic vs authentic media
- Ethical and policy analysis
- Proposed framework for responsible AI deployment
